Authentication

Requests are authenticated via the Authorization header using the Bearer scheme.

BASH 
curl https://api.neuralnet.ai/v1/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{ "model": "gpt-4-turbo" }'
Security Warning

Your API keys carry full privileges. Do not commit them to public repositories or expose them in client-side code.

Making Requests

Unified interface for chat completions. Compatible with OpenAI SDKs.

const response = await fetch('https://api.neuralnet.ai/v1/chat/completions', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: 'gpt-4-turbo',
    messages: [{ role: 'user', content: 'Hello world' }]
  })
});
